# AI-SystemAssist Website

This is a modern, dark-themed landing page for the AI-SystemAssist tool.

## Files

- `index.html` - Main page
- `styles.css` - Additional styles
- `script.js` - Interactive demo logic

## Usage

You can serve this folder using any static file server:

```bash
# Python
python -m http.server 8080

# Node
npx serve

# Or just open index.html in your browser
```

## Features

- Copyable `pip install AI-SystemAssist` command
- Direct PyPI button
- Interactive demo simulating command execution
- Modern dark UI with Tailwind CSS
- Fully responsive

## Hosting

This folder is ready to be deployed to:
- GitHub Pages
- Vercel
- Netlify
- Cloudflare Pages
- Any static host

## Customization

- Update PyPI links if needed
- Customize colors in the `<style>` tag
- Extend the interactive demo in `script.js`
